USUARIOS

OBTENER USUARIOS

  • DESCRIPCIÓN: Obtener todos los usuarios, filtrar por valores o obtener usuarios concretos.
  • URL: http://localhost/webservice2/webservice.php?usuario=1
  • MÉTODO: GET
  • VARIABLES
    1. NULL: Devuelve todos los usuarios
    2. nombre: Devuelve los usuarios con el nombre especificado
    3. edad: Devuelve todos los usuarios cno la edad especificada
